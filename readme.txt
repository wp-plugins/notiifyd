== Description == 

The Super Simple Notices plugin allows you to add beautiful, configurable (sticky) notices to your WP installation in seconds. This plugin is equipped with a color picker for easy background and text coloring as well as a selection of awesome Google fonts - guaranteed to bring life to your notices and get you noticed!  

= Features =

	√ Choose to display notices in header or footer
	√ Choose to display notices on homepage only or all pages
	√ Equipped with a colorpicker (easily change background and text color)
	√ Equipped with a selection of Google fonts - 30 fonts included that has minimal impact on page load time
	√ Allows you to link your notice (internal or external URLs)
	√ Users are able to close the notice box at any given time

== Installation ==

1. Download the .zip file
2. Upload and extract the contents of the zip file to your wp-content/plugins/folder
3. Activate the plugin from your WP-admin / Plugins
4. Enjoy!
